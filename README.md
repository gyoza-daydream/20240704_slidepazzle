# スライドパズル

## 概要

React.js + TypeScriptによるスライドパズル実装です。
マウスクリックで移動します。

このアプリは[Create React App](https://github.com/facebook/create-react-app)で作成されました。
TypeScriptを有効にしています。


### reactのメモ: `npm start`

開発モードでローカルサーバを起動します。
[http://localhost:3000](http://localhost:3000)


### reactのメモ: `npm run build`

`build` フォルダにビルドします。


### ライセンス

CC0です。
